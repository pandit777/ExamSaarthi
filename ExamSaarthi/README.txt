ExamSaarthi Android Studio Project (ready to import)

How to use:
1) Download this ZIP and open Android Studio.
2) Choose 'Open an existing project' and select the folder 'ExamSaarthi_Project'.
3) Put your logo image at:
   app/src/main/res/drawable/logo.png
   (or the appropriate drawable- density folders)
4) Add launcher icons under mipmap-* folders (ic_launcher). You can create them using Android Studio Image Asset.
5) Build -> Build Bundle(s) / APK(s) -> Build APK(s) to generate an APK.
6) For a signed release APK:
   - Build -> Generate Signed Bundle / APK -> follow steps to create/sign key.
Notes:
- This project uses Android Gradle Plugin 7.4.2 and compileSdkVersion 33.
- If Android Studio asks to update Gradle plugin/wrapper, accept or follow prompts.
